This piece was performed at Shapeshifter Lab in Brooklyn, and uses graphic conduction coded by Emma using Processing, as well as collective improvisation.

This album was a research project which used a variety of digital processes within the composition, including audio software programming and the use of microcontrollers. More info on specific processes can be found below.
A general abstract for each song can be found on this page. A much more in depth paper on how each song was composed and the software used can be found here. The code used for composition can be found with explanations on another page of this site here.
